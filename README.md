## Github User Finder 🔎

A Simple web app to search for any github user and display their profile information and Repositories.Built with *HTML*, *CSS* and *JavaScript* using the Github REST API.

-------------------------------------

##  Features 🚀

- Search GitHub users by username
- Display profile details:
   - Avatar, name, bio, location, join date
   - Followers, following, public repositories
   - Company, blog, Twitter link
- Show latest repositories with:
   - Name and description
   - Language, stars, forks
   - Last updated date
- Error handling for invalid usernames
- Responsive design

--------------------------------------------

##  Technologies 🛠️

- HTML
- CSS (Flexbox, responsive design)
- Vanilla JavaScript (ES6+)
- GitHub REST API
